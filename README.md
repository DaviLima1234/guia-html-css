## O que é HTML?

HTML (*HyperText Markup Language*) é a linguagem usada para estruturar páginas web.
Ela organiza títulos, parágrafos, imagens, links, tabelas e outros elementos.

### Principais pontos:
- É uma linguagem de marcação, não de programação.
- Define a estrutura da página.
- Trabalha com **tags**, como `<p>`, `<h1>`, `<a>`, `<img>`, etc.

---

## O que é CSS?

CSS (*Cascading Style Sheets*) é a linguagem responsável por estilizar páginas HTML.

### Principais pontos:
- Controla cores, tamanhos, fontes e posicionamento.
- Torna o visual da página mais bonito e organizado.
- Funciona junto com HTML, mas em arquivos separados.

## Vantagens

- Fácil aprendizado
- Alta compatibilidade com navegadores
- Base fundamental para quem quer aprender JavaScript

## Desvantagens

- Sem CSS, as páginas ficam simples demais
- HTML não é linguagem de programação (não executa lógica)
- Requer estudo de boas práticas para páginas grandes
